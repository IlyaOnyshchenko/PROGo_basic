Сумма последних трех цифр
Дано натуральное число. Найдите сумму последних трех цифр.

Формат входных данных
На вход дается натуральное число, не превосходящее 10^9 

Формат выходных данных
Выведите одно целое число - сумму последних трех цифр.

Sample Input 1:

65421

Sample Output 1:

7

Sample Input 2:

25

Sample Output 2:

7

Sample Input 3:

951

Sample Output 3:

15
