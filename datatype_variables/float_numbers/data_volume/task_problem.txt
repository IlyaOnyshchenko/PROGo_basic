Объем информации

Как мы знаем из курса информатики, минимальной единицей измерения количества информации является бит. 
Но есть также и другие единицы измерения, которые служат для записи большого количества информации.

Формат входных данных
На вход подается одно целое число - количество бит.

Формат выходных данных
Переведите данное количество информации в килобайты.

Примечание: заметьте, что ответ может быть вещественным числом.

Sample Input 1:

8192

Sample Output 1:

1

Sample Input 2:

12288

Sample Output 2:

1.5

Sample Input 3:

4096

Sample Output 3:

0.5
