Сумма signsign
Найдите значение z=sign(a)+sign(b)z=sign(a)+sign(b), где
sign(x)={ −1, x<0 
0,x=0 
1,x>0}
Входные данные
Вводится строка, в которой через пробел записаны два целых числа a и b.
Выходные данные
Необходимо вывести значение z.
Примечание: данную задачу предполагается решить с помощью функций.

Sample Input 1:

8 11

Sample Output 1:

2

Sample Input 2:

8 -11

Sample Output 2:

0
