Максимум двух массивов
Напишите функцию для нахождения максимального элемента массива.
Необходимо вычислить произведение двух максимальных элементов массивов a и b.
 
Входные данные
Первая строка входных данных содержит в себе одно натурально число n (1≤n≤1000) - количество элементов в массиве a.
Вторая строка содержит элементы массива a, записанные через пробел. 
Третья строка входных данных содержит в себе одно натурально число m (1≤m≤1000) - количество элементов в массиве b.
Четвертая строка содержит элементы массива b, записанные через пробел. 
Элементы массивов по модулю не превосходят 1000.
 
Выходные данные
Выведите число равное произведению максимального элемента массива a на максимальный элемент массива b.
 
Примечание: предполагается задачу решить с помощью функции.

Sample Input 1:

3
1 10 100
5
-1 15 65 -5 4

Sample Output 1:

6500

Sample Input 2:

4
-100 5 50 423
3
500 123 1000

Sample Output 2:

423000
