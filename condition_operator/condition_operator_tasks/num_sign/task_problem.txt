В математике функция sign(x) (знак числа) определена так: 
sign(x) = 1, если x > 0, 
sign(x) = -1, если x < 0, 
sign(x) = 0, если x = 0.
Говоря простым языком:

Если вводимое число меньше нуля, то выводим −1
Если число равно нулю, то выводим 0
Если число больше нуля, то выводим 1

Формат входных данных

На вход подается одно целое число x, по модулю не превышающее 10000.

Формат выходных данных
Для данного числа x выведите значение sign(x).

Примечания:

Чтобы сравнить на равенство два значения, используйте два знака "равно" (==);

Sample Input 1:

15

Sample Output 1:

1

Sample Input 2:

-5

Sample Output 2:

-1
