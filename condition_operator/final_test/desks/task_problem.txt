Парты
В некоторой школе решили набрать три новых класса и оборудовать кабинеты для них новыми партами. За каждой партой может сидеть два учащихся. Известно количество учащихся в каждом из трех классов. Выведите наименьшее число парт, которое нужно приобрести для них.

Не забывайте, что каждый класс сидит в своём кабинете, соответственно, количество парт для каждого класса должно высчитываться отдельно от остальных. 

Входные данные

Программа получает на вход три натуральных числа: количество учащихся в каждом из трех классов. Каждое число не превышает 10^5.

Выходные данные

Выведите наименьшее число парт, которое нужно приобрести для учащихся.

Sample Input 1:

10
15
24

Sample Output 1:

25

Sample Input 2:

1
1
1

Sample Output 2:

3
