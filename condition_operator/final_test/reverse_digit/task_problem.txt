Обратное число
Напишите программу, которая считывает с клавиатуры одно число и выводит обратное ему. Обра́тное число́ (обратное значение, обратная величина) к данному числу x — это число, умножение которого на x даёт единицу.
Если при этом введённое с клавиатуры число – ноль, то вывести «Обратного числа не существует» (без кавычек).

Формат входных данных 
На вход программе подается одно вещественное число.

Формат выходных данных
Программа должна вывести вещественное число обратное данному, либо текст в соответствии с условием задачи.

Sample Input 1:

2.5

Sample Output 1:

0.4

Sample Input 2:

-55.6

Sample Output 2:

-0.017985611510791366

Sample Input 3:

0

Sample Output 3:

Обратного числа не существует
