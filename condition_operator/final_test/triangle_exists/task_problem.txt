Существует ли треугольник?
По данным сторонам треугольника, определите, существует ли невырожденный треугольник.

Вырожденный треугольник — это треугольник, у которого все три вершины лежат на одной прямой. Все остальные треугольники называются невырожденными. 

Свойство:

Невырожденный треугольник существует, когда сумма любых двух сторон строго больше третьей стороны.

Входные данные

Даны три натуральных числа a,b,c, каждое из которых не превышает 10000.

Выходные данные

Если существует невырожденный треугольник, выведите строку "YES", иначе выведите строку "NO". Когда треугольник не существует, то нужно вывести "NO".

Sample Input 1:

3
4
5

Sample Output 1:

YES

Sample Input 2:

1
2
3

Sample Output 2:

NO

Sample Input 3:

4
5
3

Sample Output 3:

YES
