Симметричное число
Дано четырехзначное число. Определите, является ли его запись симметричной. Число называется симметричным, если оно одинаково читается справа налево и слева направо.

Формат входных данных
Вводится натуральное число, не большее 9999.

Формат выходных данных
Выведите "YES", если число симметричное,  иначе "NO".

Sample Input 1:

1554

Sample Output 1:

NO

Sample Input 2:

2552

Sample Output 2:

YES
