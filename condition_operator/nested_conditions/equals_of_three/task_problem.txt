Количество равных из трех
Определите среди трех целых чисел количество совпадающих.

Входные данные
Даны три целых числа, каждое из которых не превышает 10^6.

Выходные данные
Программа должна вывести одно из чисел: 3 (если все совпадают), 2 (если два совпадают) или 0 (если все числа различны).

Sample Input 1:

1
2
3

Sample Output 1:

0

Sample Input 2:

1
1
2

Sample Output 2:

2

Sample Input 3:

1
2
1

Sample Output 3:

2

Sample Input 4:

2
1
1

Sample Output 4:

2

Sample Input 5:

2
2
2

Sample Output 5:

3
