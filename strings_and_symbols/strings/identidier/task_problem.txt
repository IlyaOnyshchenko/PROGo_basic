Идентификатор
Определить, является ли введенное слово идентификатором. Идентификатор должен начинаться с английской буквы (в любом регистре) или знака подчёркивания, а так же введенное слово НЕ должно содержать других символов, КРОМЕ букв английского алфавита (в любом регистре), цифр и знака подчеркивания.
   
Входные данные
На вход программе подается одна строка.
Выходные данные
Если строка является идентификатором, необходимо вывести "YES", иначе - "NO".

Sample Input 1:

BeeGeek

Sample Output 1:

YES

Sample Input 2:

1024BeeGeek

Sample Output 2:

NO

Sample Input 3:

_Dada123__

Sample Output 3:

YES

Sample Input 4:

132132Adda___

Sample Output 4:

NO
