Цифры в тексте
Дан текст, составленный из букв латинского алфавита и цифр. Напечатать все имеющиеся в нем цифры.
 
Входные данные
На вход программе подается текст, составленный из букв латинского алфавита и цифр, длина которого не превосходит 10001000.
Выходные данные
Выведите все имеющие цифры в тексте на одной строке через пробел.

Sample Input 1:

Ronaldo7

Sample Output 1:

7

Sample Input 2:

jkshf2kj2kj3kj4lk7

Sample Output 2:

2 2 3 4 7
