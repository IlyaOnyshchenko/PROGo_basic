Инициалы
По данным ФИО сформируйте строку, содержащую фамилию с инициалами. Например, по ‘Ivanov Ivan Ivanovich’ получите "Ivanov I.I.".

Входные данные
На вход программе подаются 3 строки - фамилия имя отчество.
Выходные данные
Выведите строку, содержащую фамилию с инициалами.

Sample Input:

Ivanov
Ivan
Ivanovich

Sample Output:
Ivanov I.I.
