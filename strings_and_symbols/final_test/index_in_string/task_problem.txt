Индексация в строке
Даны две строки. Верно ли, что первая строка начинается на ту же букву, на которую заканчивается вторая строка? Регистр букв имеет значение, то есть 'a' и 'A' считаются разными буквами.
 
Входные данные
На вход программе подаются 22 строки. Длины данных строк не превосходят 10001000.
Выходные данные
Выведите "YES", если первая строка начинается на ту же букву, на которую заканчивается вторая строка, иначе выведите "NO".

Sample Input 1:

abaca
jdsfa

Sample Output 1:

YES

Sample Input 2:

abc
abc

Sample Output 2:

NO
