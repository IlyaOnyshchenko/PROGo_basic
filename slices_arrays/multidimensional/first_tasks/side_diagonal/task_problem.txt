Побочная диагональ
Дано число n,n≤100n,n≤100. Создайте массив n×nn×n и заполните его по следующему правилу:
•	числа на диагонали, идущей из правого верхнего в левый нижний угол, равны 1;
•	числа, стоящие выше этой диагонали, равны 00;
•	числа, стоящие ниже этой диагонали, равны 22.
Входные данные
Программа получает на вход число nn.
 
Выходные данные
Необходимо вывести полученный массив. Числа разделяйте одним пробелом.
Например, если n = 3, то надо вывести

0 0 1
0 1 2
1 2 2

Sample Input 1:

3

Sample Output 1:

0 0 1
0 1 2
1 2 2

Sample Input 2:

1

Sample Output 2:

1

Sample Input 3:

2

Sample Output 3:

0 1
1 2
