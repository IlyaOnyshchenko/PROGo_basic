Поменять местами первый минимум и последний максимум
Дан массив, состоящий из целых чисел. Напишите программу, которая меняет местами первый минимальный и последний максимальный элементы массива. Индексация начинается с нуля.
Входные данные
Сначала задано число NN — количество элементов в массиве (1≤N≤10001≤N≤1000). Далее через пробел записаны NN чисел — элементы массива. Массив состоит из целых чисел, каждое из которых не превышает 1000010000.
Выходные данные
Необходимо вывести массив, в котором поменяны местами первый минимальный и последний максимальный элементы массива.
Например:
1 2 1 1 2
Первый минимальный это 1 с индексом 0. Последний минимальный это 1 с индексом 3.
Первый максимальный 2 с индексом 1. Последний максимальный 2 с индексом 4.

Sample Input 1:

5
1 2 3 1 3

Sample Output 1:

3 2 3 1 1

Sample Input 2:

3
2 2 1

Sample Output 2:
2 1

Sample Input 3:

2
1 2

Sample Output 3:
2 1
