Поиск - 2
Дан массив, состоящий из целых чисел. Напишите программу, которая выводит те элементы массива, которые кратны трем.

Входные данные

Сначала задано число N — количество элементов в массиве (1≤N≤100). Далее через пробел записаны N чисел — элементы массива. Массив состоит из целых чисел, каждое из которых по модулю не превышает 10^5.
Выходные данные

Необходимо вывести все элементы массива, кратные трем.

Sample Input:

7
1 2 3 4 5 6 7

Sample Output:

3 6
