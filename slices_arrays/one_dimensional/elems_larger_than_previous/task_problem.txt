Количество элементов, больших предыдущего
Дан массив, состоящий из целых чисел. Напишите программу, которая подсчитает количество элементов массива, больших предыдущего (левого соседа).
Входные данные
Сначала задано число NN — количество элементов в массиве (1≤N≤1001≤N≤100). Далее через пробел записаны NN чисел — элементы массива. Массив состоит из целых чисел, каждое из которых по модулю не превышает 109109.
Выходные данные
Необходимо вывести единственное число — количество элементов массива, больших предыдущего элемента.
 
Объяснение первого теста:
Дан массив 1 3 2 4 5. Давайте посчитаем сколько элементов больше предыдущего. 
Первый элемент массива 1. У него нет предыдущего элемента. Значит пропускаем.
Второй элемент массива 3. Предыдущий элемент равен 1. 3 больше 1, значит данный элемент подходит. 
Третий элемент массива 2. Предыдущий элемент равен 3. 2 НЕ больше 3, значит данный элемент НЕ подходит. 
Четвертый элемент массива 4. Предыдущий элемент равен 2. 4 больше 2, значит данный элемент подходит. 
Пятый элемент массива 5. Предыдущий элемент равен 4. 5 больше 4, значит данный элемент подходит. 
Следовательно, 3 элемента подходят.

Sample Input 1:

5
1 3 2 4 5

Sample Output 1:

3

Sample Input 2:

5
1 2 3 4 5

Sample Output 2:

4

Sample Input 3:

5
5 4 3 2 1

Sample Output 3:

0
