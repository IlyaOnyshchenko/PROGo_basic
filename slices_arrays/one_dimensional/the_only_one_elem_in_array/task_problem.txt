Единственные элементы
Дан список. Выведите те его элементы, которые встречаются в списке только один раз.
Входные данные
Сначала задано число NN — количество элементов в массиве (1≤N≤100001≤N≤10000). Далее через пробел записаны NN чисел — элементы массива. Массив состоит из целых чисел, каждое из которых по модулю не превышает 1000010000.
Выходные данные
Необходимо вывести на одной строке через пробел элементы, которые встречаются в списке только один раз. Элементы нужно выводить в том порядке, в котором они встречаются в списке.

Sample Input 1:

5
1 2 1 3 1

Sample Output 1:

2 3

Sample Input 2:

4
1 3 3 1

Sample Output 2:
