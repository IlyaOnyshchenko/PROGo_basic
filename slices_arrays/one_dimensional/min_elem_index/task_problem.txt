Индекс минимального элемента
Дан массив, состоящий из целых чисел. Напишите программу, которая выведет индекс минимального элемента массива. Индексация начинается с нуля.
Входные данные
Сначала задано число NN — количество элементов в массиве (1≤N≤10001≤N≤1000). Далее через пробел записаны NN различных чисел — элементы массива. Массив состоит из целых чисел, каждое из которых по модулю не превышает 10000001000000.
Выходные данные
Необходимо вывести индекс минимального элемента массива.

Sample Input 1:

3
3 1 2

Sample Output 1:
1

Sample Input 2:

5
-100 5 54 524 5422

Sample Output 2:
0

Sample Input 3:
1
0

Sample Output 3:
0
