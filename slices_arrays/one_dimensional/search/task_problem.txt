Поиск-1
Дан массив, состоящий из целых чисел. Напишите программу, которая выведет элементы массива, индексы которых кратны трем (0, 3, 6...). 
Элементы выведите на одной строке через пробел.

Входные данные

Сначала задано число N — количество элементов в массиве (1≤N≤100). Далее через пробел записаны N чисел — элементы массива. 
Массив состоит из целых чисел, каждое из которых по модулю не превышает 10^5.

Выходные данные

Необходимо вывеcти элементы массива, индексы которых кратны трем.

Sample Input:

7
1 3 2 4 5 0 8

Sample Output:

1 4 8
