Сумма с условием
Найдите сумму введенных чисел, которые кратны 2, но не кратны 3.

Входные данные
Вводится натуральное число N(N≤10^5), обозначающее количество вводимых чисел. Затем N чисел, каждое из которых по модулю не превышает 2∗10^4.

Выходные данные
Вычислите сумму введенных чисел, кратных 2 , но не кратных 3.

Sample Input 1:

3
2
3
4

Sample Output 1:

6

Sample Input 2:

1
1

Sample Output 2:

0
