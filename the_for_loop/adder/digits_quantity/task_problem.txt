Количество чисел
По данным числам, определите количество чисел, оканчивающиеся на 0.  

Входные данные
Вводится натуральное число N(N≤10^5) а затем N чисел, каждое из которых по модулю не превышает 2∗10^6

Выходные данные
Вычислите количество чисел, которые оканчиваются на 0.

Sample Input 1:

3
10
25
40

Sample Output 1:

2

Sample Input 2:

2
15
37

Sample Output 2:

0
