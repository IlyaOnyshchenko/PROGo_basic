Обратное число
Дано натуральное число N. Переставьте цифры числа в обратном порядке.

Входные данные

Вводится натуральное число N, не превышающее 10^9.

Выходные данные

Выведите строку, полученную с помощью перестановки цифр числа N в обратном порядке.

Sample Input 1:

125

Sample Output 1:

521

Sample Input 2:

600

Sample Output 2:

006
