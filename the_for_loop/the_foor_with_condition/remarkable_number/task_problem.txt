Замечательное число
Дано натуральное число N. Определите является ли данное число замечательным. 
Число называется замечательным, если оно делится на сумму своих цифр без остатка.

Входные данные
Вводится натуральное число N, не превышающее 10^9.

Выходные данные
Выведите "YES", если число N замечательное, в противном случае "NO".

Sample Input 1:

1

Sample Output 1:

YES

Sample Input 2:

7

Sample Output 2:

YES

Sample Input 3:

10

Sample Output 3:

YES

Sample Input 4:

11

Sample Output 4:

NO
