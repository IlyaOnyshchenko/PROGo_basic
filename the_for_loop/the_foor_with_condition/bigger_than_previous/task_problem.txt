Количество элементов, которые больше предыдущего
Последовательность состоит из натуральных чисел и завершается числом 0. 
Определите, сколько элементов этой последовательности больше предыдущего элемента. Числа, следующие за числом 0, считывать не нужно.

Входные данные
Вводится последовательность натуральных чисел, оканчивающаяся числом 0 (само число 0 в последовательность не входит). 
Числа не превышают 1000.

Выходные данные
Выведите количество  элементов последовательности больших предыдущего элемента.

Sample Input:

1
2
3
0
6

Sample Output:

2
