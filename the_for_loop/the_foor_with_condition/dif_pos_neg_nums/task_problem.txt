Разность положительных и отрицательных чисел
Последовательность состоит из целых чисел и завершается числом 0. 
Определите разность количества положительных и отрицательных чисел последовательности.

Входные данные

Вводится последовательность целых чисел, оканчивающаяся числом 0 (само число 0 в последовательность не входит). 
Числа по модулю не превышают 1000.

Выходные данные

Выведите разность количества положительных и количества отрицательных чисел, 
то есть результат операции "количество положительных чисел" минус "количество отрицательных чисел".

Sample Input 1:

2
3
-8
9
0
54
23

Sample Output 1:

2

Sample Input 2:

3
-3
0
2

Sample Output 2:

0
