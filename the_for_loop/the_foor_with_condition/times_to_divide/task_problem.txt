Сколько раз делится
По данному натуральному числу a, определите, сколько раз a делится на 3 нацело (без остатка).

Входные данные
Вводится натуральное число a, не превышающее 10^6.

Выходные данные

Выведите сколько раз a делится на 3 нацело (без остатка).

Sample Input 1:

54

Sample Output 1:

3

Sample Input 2:

55

Sample Output 2:

0
