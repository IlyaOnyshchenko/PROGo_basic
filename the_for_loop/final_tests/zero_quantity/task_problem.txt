Количество нулей
По данным числам, определите количество чисел, которые равны нулю.  

Входные данные
Вводится натуральное число N(N≤10^5), а затем N чисел, каждое из которых не превышает 2∗10^4.

Выходные данные
Выведите количество чисел, которые равны нулю.

Примечание: для лучшего понимания задачи, рассмотрите предложенные примеры. 

Sample Input 1:

4
5
2
0
4

Sample Output 1:

1

Sample Input 2:

4
502
980
0
45

Sample Output 2:

1
