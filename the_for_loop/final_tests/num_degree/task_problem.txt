Степень числа
Для целых чисел a и n вычислить a^n.

Входные данные

Вводятся два целых числа a и n, при этом 1 <= a <= 100, 0 <= n <= 20

Выходные данные

Выводится одно число a^n.

Примечание: гарантируется, что ответ не больше 2⋅10^9. НЕ используйте Math.Pow!

Sample Input 1:

2
3

Sample Output 1:

8

Sample Input 2:

3
1

Sample Output 2:

3

Sample Input 3:

5
0

Sample Output 3:

1
