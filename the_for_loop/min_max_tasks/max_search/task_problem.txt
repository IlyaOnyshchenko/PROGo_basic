Максимум
Найдите максимальное из введенных чисел.

Входные данные

Вводится натуральное число N(N≤10^5), а затем N целых чисел, каждое из которых по модулю не превышает 10^6.

Выходные данные

Выведите максимальное из введенных чисел.

Sample Input 1:

3
3
1
2

Sample Output 1:

3

Sample Input 2:

3
-100
0
100

Sample Output 2:

100

Sample Input 3:

3
1
2
3

Sample Output 3:

3
