Количество четверок
Дано натуральное число N. Найдите количество четверок среди всех цифр числа N.

Входные данные
Вводится натуральное число N, не превышающее 10^9.

Выходные данные
Выведите одно число - количество четверок среди всех цифр числа N.

Sample Input 1:

425

Sample Output 1:

1
Sample Input 2:

244

Sample Output 2:

2
